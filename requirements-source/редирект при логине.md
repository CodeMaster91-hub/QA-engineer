# Редирект на приватные домены

## Цель

Продлить срок жизни домена, открывать его только проверенным клиентам и не показывать обычным.



## Требования

- [x] Добавить "тип домена" - "Приватный домен" в разделе /admin/app/mirror/list
- [x] Добавить 10 зеркалам тип "Приватный домен" 
- [x] При автологине проверенных пользователей направлять на приватный домен

## Реализация

Добавить в административной панели **новый тип домена** "Приватный домен", по которому будет определяться зеркало для редиректа доверенных пользователей.

> Подробнее про раздел "Зеркала" описано на странице [Описание зеркал (COM-версия)](/pages/viewpage.action?pageId=633628524)

При автологине или при авторизации, **проверять категорию пользователя** и редиректить проверенных на приватный домен.

1. **user_category.id = 32** (title = "Без задержки на создание купона"), то **редиректнуть на домен из списка** "Приватные домен" (если доменов в этой категории нет, то оставить на месте)
2. **Любая другая категория** - оставить на месте
3. **Если клиент уже на приватном домене** - оставить на месте

### Критерии выбора домена:

1. Домен имеет лейбл "приватный клиентский домен"
2. Домен активен
3. Домен для web

## Тестирование

### Кейс 1

**Условия:**
- В административной панели присвоить тип "Приватный домен" для любого домена
- Присвоить категорию "Без задержки на создание купона" тестовому пользователю

**Действия:**
1. Тестовый пользователь авторизуется на домене 1 без типа "Приватный домен", сохраняется токен авторизации для дальнейшей проверки
2. Тестовый пользователь открывает в соседней вкладке домен 2 без типа "Приватный домен"

**Ожидаемый результат:** Происходит редирект на домен с типом "приватный домен", пользователь авторизован.

### Кейс 2

**Условия:**
- В административной панели домен с типом "Приватный домен" заблокировать

**Действия:**
1. Тестовый пользователь авторизуется на домене 1 без типа "Приватный домен", сохраняется токен авторизации для дальнейшей проверки
2. Тестовый пользователь открывает в соседней вкладке домен 2 без типа "Приватный домен"

**Ожидаемый результат:** Пользователь авторизован, остался на домене 2.

**Действия:**

2. Тестовый пользователь открывает в соседней вкладке домен 2 без типа "Приватный домен"
